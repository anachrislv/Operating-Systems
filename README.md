# **Operating Systems Course - Labs (2021-2022 Spring Semester, ECE NTUA)**

This repository contains lab projects for the **Operating Systems** course at **ECE, NTUA** in the **2021-2022 Spring Semester**.

## **Authors**
- **Anastasia Christina Liva**
- **George Mystriotis**

## **Contents**
- **Lab 1**: Introduction to programming environment and object file linking.
- **Lab 2**: Process management, tree creation, and inter-process communication using signals and pipes.
- **Lab 3**: Synchronization using POSIX threads and semaphores.
- **Lab 4**: Virtual memory management and inter-process synchronization.
